🚀 Live Crypto Dashboard

It is a lightweight web dashboard that displays real-time cryptocurrency data.
It shows the top 10 cryptocurrencies, their prices, 24h changes, and global market stats using live data from CoinGecko.

📡 API Used
CoinGecko Public API

Market Data:
https://api.coingecko.com/api/v3/coins/markets

Global Data:
https://api.coingecko.com/api/v3/global
